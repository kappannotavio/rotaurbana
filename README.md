# 🚍 Sistema de Rastreamento de Transporte Universitário

## 📌 Sobre o projeto
Este projeto foi desenvolvido como parte da disciplina de Projeto Integrador do curso de Ciência da Computação.

O sistema tem como objetivo permitir o **rastreamento em tempo real de veículos de transporte universitário**, facilitando o acesso à localização do ônibus pelos passageiros e melhorando a organização do transporte.

---

## 🎯 Problema
Atualmente, passageiros enfrentam dificuldades como:
- Falta de informação sobre a localização do ônibus  
- Atrasos sem aviso prévio  
- Falta de previsibilidade no trajeto  

---

## 💡 Solução
O sistema permite:
- Envio da localização em tempo real do motorista  
- Visualização do ônibus em um mapa  
- Acompanhamento do trajeto pelos usuários  

---

## 🛠️ Tecnologias utilizadas
- Java (Spring Boot)  
- HTML, CSS e JavaScript  
- Leaflet (mapas)  
- REST API  
- Git e GitHub  

---

## ⚙️ Funcionalidades
- 📍 Envio de coordenadas (latitude e longitude) via API  
- 🗺️ Exibição do ônibus em mapa interativo  
- 🔄 Atualização em tempo real da localização  
- 📡 Comunicação entre backend e frontend
